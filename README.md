# Sakeenah Clothing Brand Website

A modern, multi-page website for the Sakeenah clothing brand.

## Features

- **Home Page**: Welcome section with featured products
- **Products Page**: Full product catalog with add to cart functionality
- **Individual Product Pages**: Detailed product information with image pagination
- **About Page**: Brand story and mission
- **Shopping Cart**: Full cart functionality with dropdown and checkout
- **"SELLING FAST" Badges**: Eye-catching product badges
- **Sale Pricing**: Display original and sale prices
- **SMS Popup**: First-time visitor popup with 10% discount offer
- **Email/SMS Signup**: "Join Our Ummah" footer signup
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern Design**: Clean, minimalist aesthetic inspired by premium streetwear brands

## Pages

1. **index.html** - Homepage with hero section and featured products
2. **products.html** - Complete product catalog
3. **about.html** - About page with brand information
4. **cart.html** - Dedicated cart page with recommendations
5. **product-classic-tee.html** - Stripe Shirt product detail page
6. **product-elegant-dress.html** - Islamic Jeans product detail page

## How to Use

1. Simply open `index.html` in your web browser
2. Navigate between pages using the navigation menu
3. Click on any product to view its detailed information
4. Click "Add to Cart" buttons on products to see the functionality

## Customization

- **Colors**: Edit the color values in `styles.css` (currently using basic grays and blacks)
- **Products**: Add or modify products in `products.html`
- **Content**: Update text content in any HTML file
